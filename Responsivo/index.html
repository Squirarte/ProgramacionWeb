Diseño Responsivo

Responsive Web Design es la técnica que nos permite crear sitios adaptables a las condiciones del ordenador o dispositivo desde donde se van a acceder, sobre todo en lo que tiene relación con la pantalla del sistema donde se están visualizando.
Todo comenzó con Ethan Marcotte cuando introdujo el término Responsive Web Design y publicó un libro con el mismo nombre.
Las páginas web no deben omitir información para adaptarse a dispositivos más pequeños.

¿De que trata?

El diseño web responsivo se trata de usar HTML Y CSS
para cambiar el tamaño, ocultar, reducir o ampliar automáticamente un sitio web, para que se vea bien en todos los dispositivos.Las páginas optimizadas para una variedad de dispositivos deben
incluir una Meta etiqueta de ventana gráfica en el encabezado del documento. Una Meta etiqueta de ventana gráfica le da al navegador
instrucciones sobre cómo controlar las dimensiones y la escala de la página.

Imagenes

Una imagen tiene dimensiones fijas y si es más grande que la ventana gráfica, se formará una barra de desplazamiento.

Una forma habitual de solucionar este problema es asignar a todas las imágenes una max-width de 100%. Esto hará que la imagen se encoja para ajustarse al espacio que tiene, en caso de que el tamaño de la ventana gráfica sea más pequeño que la imagen

Características

  • Los layouts e imágenes son fluidos y se adaptan a cada pantalla.
  • La distribución de los diferentes elementos (layout) y las imágenes se reacomodan para cada dispositivo.
  • Se usa código media-queries de CCS3, para permitir la fluidez de las imágenes y el contenido en general.
  • Permite compartir contenidos de manera rápida y natural.

Diseño

Dado que las dimensiones y el ancho de la pantalla en píxeles CSS varían ampliamente entre dispositivos (por ejemplo, entre teléfonos y tabletas, e incluso entre diferentes teléfonos), el contenido no debe depender de un ancho de ventana en particular para renderizarse bien.
Existe un método llamado flexbox para mostrar elementos como una sola fila o envueltos en varias filas a medida que disminuye el espacio disponible.
